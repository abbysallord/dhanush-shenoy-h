Create a `.env` file in the project root:


> **Getting a Formspree endpoint:** Go to [formspree.io](https://formspree.io), create a free form, and copy the endpoint URL. Without this, the contact form will silently fail.

## `// design philosophy`

This portfolio deliberately avoids UI libraries. Here's why:

1. **CSS is a superpower** â€” Every visual detail is precisely controlled via custom properties and hand-written rules
2. **Animations tell stories** â€” GSAP timelines sequence every element's entrance so the page feels alive, not dumped
3. **Performance-first** â€” Lenis + GSAP ticker synchronization prevents jank. `IntersectionObserver` for lazy reveals instead of scroll listeners
4. **Content is config** â€” Changing a job or project requires zero JSX knowledge â€” just edit the data object at the top of the file
